# Tech Stack: Munchy

## Frontend
- **Framework:** Next.js 16 (App Router)
- **Library:** React 19
- **Language:** TypeScript
- **UI Components:** Chakra UI v3
- **Animations:** Framer Motion
- **Data Visualization:** Recharts
- **Barcode Scanning:** html5-qrcode

## Backend & Infrastructure
- **BaaS:** Supabase (Auth, Database, Storage)
- **Runtime:** Bun
- **AI/LLM:** OpenAI (@openai/agents)
- **APIs:** USDA FoodData Central API, OpenFoodFacts

## Tooling
- **Linting:** ESLint
- **Formatting:** Prettier
- **Validation:** Zod
- **Documentation:** Markdown
